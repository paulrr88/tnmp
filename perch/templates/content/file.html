Instructions:

- "Text to display in link to file" field: this is what the link will say, for instance "OFSTED report document", which would then output a link saying just that, which the user can then click to be taken to the download in question.

- "File" field: this area is where you can select your file. To do this you will need to:
- Click "Select or upload a file"
- In the resulting Select an Asset area that pops up, either:
- Click an existing item that you want to link to and then click "Use Selected", or
- Click the "+Add Asset" button in the top right, and then either drop a file into the indicated box area or click the box area to select the file from your computer.
- Click the file that you just added, now in a little box in the "Select an Asset" area
- Click "Use Selected" in the top right of the "Select an Asset" section ("Use Selected" should be highlighted green if you've correctly clicked a file to be used, if not just try clicking again or "Use Selected" won't do anything)
- N.B. if you enter this area by accident and want to go back, click the grey (X) icon in the far top right of the screen.
- PDF format for documents is preferable if you can manage it as more people can read this format than items created in Word etc!

- "Download type" field: this is the file type in question which is displayed after the link to tell the user what it is that they are downloading. For instance, PDF or Word Document.
